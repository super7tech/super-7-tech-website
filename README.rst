React SPA Example
=================

Setup
-----

**1. Install latest Node**

Download for Mac: https://nodejs.org/en/download/

**2. Create React App**

    $ npx create-react-app

**3. Git Init**

    $ git init; git remote add origin ...

**4. Remove Unnecessary Files**

    $ rm -f README.md

Test
----

**1. Run Tests**

    $ yarn test [a(all)|w(watch)]
